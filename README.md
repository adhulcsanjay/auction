## Deploy using Docker

First, build project using node and then build docker image:
```bash
npm run build
docker build -t event-registration-ui:latest .
```
or we can download the image tar from the CI/CD pipeline.
1. Open Build > Artifacts
2. Use download button in the latest package job
3. Unzip the file
Then import the tar to the docker using
``` bash
docker load --input <TAR_FILE>
```
### Run using Docker
```bash
docker run -p 80:3000 event-registration-ui:main
```
